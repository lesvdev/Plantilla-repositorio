# 🖼️ Carpeta de Imágenes

Aquí debes guardar todas las imágenes de tu portafolio.

## Imágenes necesarias

### 1. Foto de Perfil
- **Nombre sugerido:** `mi-foto.png` o `mi-foto.jpg`
- **Tamaño recomendado:** 400x400 píxeles (cuadrada)
- **Formato:** PNG o JPG
- **Ubicación:** `images/mi-foto.png`

### 2. Escudo de la Institución
- **Nombre sugerido:** `escudo.png` o `escudo.jpg`
- **Tamaño recomendado:** 200x200 píxeles (cuadrada)
- **Formato:** PNG con fondo transparente (ideal)
- **Ubicación:** `images/escudo.png`

### 3. Obras de Arte (para página artística)
- **Nombres sugeridos:** `obra1.jpg`, `obra2.jpg`, etc.
- **Tamaño recomendado:** 1200x800 píxeles (horizontal)
- **Formato:** JPG para fotos, PNG para dibujos digitales
- **Ubicación:** Subir a GitHub y usar URL raw

## Formatos recomendados

| Uso | Formato | Razón |
|-----|---------|-------|
| Fotos | JPG | Menor tamaño, buena calidad |
| Logos/Escudos | PNG | Fondo transparente |
| Dibujos digitales | PNG | Calidad sin pérdida |
| Iconos | PNG/SVG | Escalables |

## Optimización de imágenes

Para que tu página cargue rápido:

1. **Comprime las imágenes** antes de subirlas
2. **No uses imágenes muy grandes** (máximo 2MB por imagen)
3. **Usa dimensiones apropiadas** (no subas una foto de 4000x3000 píxeles)

### Herramientas gratuitas para comprimir imágenes:
- [TinyPNG](https://tinypng.com/) - Comprime PNG y JPG
- [Squoosh](https://squoosh.app/) - De Google, muy completa
- [ImageOptim](https://imageoptim.com/) - Para Mac

## Ejemplo de uso en HTML

```html
<!-- Foto de perfil -->
<img src="images/mi-foto.png" alt="Mi foto de perfil" class="foto-perfil">

<!-- Escudo -->
<img src="images/escudo.png" alt="Escudo del colegio" class="logo-institucion">

<!-- Obra de arte desde GitHub -->
<img src="https://raw.githubusercontent.com/usuario/repo/main/images/obra1.jpg" 
     alt="Mi obra de arte" class="carrusel-imagen">
```
