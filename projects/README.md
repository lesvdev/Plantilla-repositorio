# 📁 Carpeta de Proyectos

Esta carpeta está destinada para tus proyectos futuros.

## ¿Qué puedes guardar aquí?

- 📄 Proyectos HTML adicionales
- 🐍 Scripts de Python avanzados
- 🎨 Archivos de diseño gráfico
- 📊 Proyectos de análisis de datos
- 🎮 Videojuegos creados
- 📱 Proyectos de aplicaciones
- 📝 Documentación de proyectos

## Organización sugerida

```
projects/
├── 📁 proyecto-1/
│   ├── index.html
│   ├── css/
│   └── js/
├── 📁 proyecto-2/
│   └── ...
└── 📁 proyecto-python/
    └── script.py
```

## Consejos

1. **Usa nombres descriptivos** para las carpetas
2. **Incluye un README** en cada proyecto explicando qué hace
3. **Organiza por fecha** o por materia
4. **Haz backups** de tus proyectos importantes
