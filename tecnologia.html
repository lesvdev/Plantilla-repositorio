📝 ¿Cómo agregar contenido a esta sección?

Cuando estés listo para agregar proyectos, sigue estos pasos:

  1. Abre el archivo tecnologia.html en tu editor de código
  2. Busca la sección que dice "SECCIÓN DE CONTENIDO"
  3. Reemplaza el mensaje de "en construcción" con tus proyectos
  4. Usa las mismas clases CSS que en las otras páginas para mantener el diseño
  5. Guarda los cambios y actualiza tu navegador

💡 Recuerda: Puedes copiar el código de las tarjetas de la página index.html y adaptarlo para tus proyectos de tecnología.
